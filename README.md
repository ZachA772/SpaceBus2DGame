# SpaceBus2DGame# (Unity Cross-Platform Project)

This project is a 2D space shooter developed using Unity and C#. The player controls a rocket-powered bus and must survive waves of various different enemies.

The project was designed to support multiple platforms including mobile, WebGL, and desktop.

## Features

•	Player movement
•	Shooting system
•	Multiple Levels
•	Multiple enemy types
•	Multiple Bosses
•	Power-ups
•	Collision detection
•	Sound effects
•	UI displaying score, Boss Names/Health Bars
•	Cross-platform support with Mobile, WebGL, and Desktop

## How to Run the Project in Unity

1. Open Unity Hub  
2. Click "Open Project From Disk"  
3. Select the SpaceShooterGame folder  
4. Open the scene from Assets/Scenes
5. Click Play
  

## Controls

Desktop:
• Move: WASD or Arrow Keys  
• Shoot: Left Mouse on the "Shoot" Button  

Mobile:
• Touch controls   

## Unity Version
